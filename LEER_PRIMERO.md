## 🚀 Cómo usar

### En Windows
1. Abre Explorador de archivos
2. Ve a: `C:\claude code\ml-crm-app\`
3. Haz doble click en `index.html`
4. ¡Listo! Se abre en tu navegador

### En Navegador
1. Presiona `Ctrl+L` (barra de direcciones)
2. Pega: `file:///c:/claude%20code/ml-crm-app/index.html`
3. Presiona Enter

## 📋 Qué hace la app

✅ **Dashboard** - Ve tus ventas, ganancias, productos top
✅ **Ventas** - Registra cada venta que hagas
✅ **Productos** - Agrega/edita tus productos
✅ **Inventario** - Controla tu stock
✅ **Análisis** - Tabla detallada de ganancias
✅ **Alertas** - Avisos de stock bajo
✅ **Configuración** - Exporta/importa datos

---

## 🔧 Funcionalidades

### Dashboard
- KPIs: Ganancia, unidades vendidas, ingresos, comisiones
- Gráficos automáticos
- Top 5 productos
- Ranking completo
- Alertas de stock bajo

### Ventas
- Formulario rápido para registrar ventas
- Elige producto + cantidad
- Calcula automáticamente ganancia
- Historial completo

### Productos
- Agregar productos con:
  - Precio de compra
  - Precio de venta
  - Stock inicial
  - Costo de envío
  - Tipo de comisión (% o monto fijo)
- Editar productos
- Borrar (archiva, no elimina datos)

### Datos
- Todo se guarda automáticamente en tu navegador (localStorage)
- Exporta como JSON
- Importa desde JSON o CSV

## 💡 Tips

- **Atrás los datos**: Si borras localStorage, pierdes todo. Exporta regularmente.
- **Mobile**: En celular, la navegación está abajo (swipe para ver todas)
- **Comisiones**: Puedes usar % o monto fijo
- **Stock**: Te alerta cuando stock < 25%
